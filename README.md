* 선택한 브랜드에 따라 상품 리스트 나열
* 선택한 브랜드내에 색상 필터 구현
* 각 상품별 상품이미지/브랜드/상품가/할인상품가/할인율 노출

# 👩‍💻 About Project
- 상품 api를 활용하여 쇼핑몰을 재현합니다. 
- 선택한 브랜드의 상품이 나열됩니다.
- 브랜드내에 색상을 선택하면, 선택한 색상이 포함된 상품만 나열됩니다.  
- 기존 상품가와 할인된 상품가를 계산하여 할인율을 나타냅니다. 
- 상품리스트에서 뒤로가기를 누르면, 브랜드 리스트 페이지가 노출되고, 색상 필터는 초기화됩니다.
